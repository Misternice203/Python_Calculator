# Python_Calculator
Basic CLI Calculator using Python.

## Code
[View Calculator.py](Calculator.py)

A simple command line calculator built with python, that preforms basic arithmatic operations.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Division by zero error handling
- Exponents
- Continuous calculations using while loop

## Example

Enter first number: 5
Enter second number: 3

1. Add
2. Subtract
3. Multiply
4. Divide
5. Exponentiation
Choose operation: +
Result: 8

## How to Run

1. Make sure python is installed
2. Download the repository
3. Run the file using: Calculator.py
